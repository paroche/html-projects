Dynamic Sites
Recipe Lookup Site -- Pure JS

"Menutopia" is a pure vanilla JavaScript site built using the MVC model in the course of completing Jonas Schmedtmann's "The Complete JavaScript Course 2020: Build Real Projects!" (Where it is called "forkify"). It uses an API to pull recipes from a wide variety of websites, parses the ingredient lists, allows users to track favorite recipes, adjust portions, and add recipe ingredients to their shopping list. (The purpose of the course was to teach JavaScript, not html & css, so the templates for the html and the css were mostly provided -- and the site is not responsive.)

Tour Site -- Express.js & Mongoose

"Natours" is a fictional nature tours site created in the course of taking Jonas Schemedtmann's course "Node.js, Express, MongoDB & More: The Complete Bootcamp 2020". It uses JavaScript / Node.js / Node Express / MongoDB / Mongoose for the API and backend, and Pug Templates to render the front end. It has its own API which can be accessed its data from other sites and which it also uses to handle requests from the front end user. It allows users to look at the tours, see maps of where they go, book tours through Stripe credit card processing, add reviews of tours, etc. It provides multiple levels of user security, including email verification for account creation. Not all user features have yet been implemented. The site is beautiful but not responsive. Also, it may take a few seconds for heroku to wake up and serve the site.

University Web Site -- WordPress

"Fictional University" is a, well, fictional university site built in WordPress for the Brad Schmitt' tutorial "Become a WordPress Developer: Unlocking Power with Code", where visitors can create accounts and log in, view upcoming events and blog posts, create their own blog posts, view programs, professors, and campuses (with Google Maps interface). It is currently hosted at SiteGround, until late October 2020.

Social Networking Site -- React.js

"Reactions" is a simple social media site built for Brad Schmitt's "React for the Rest of Us" course. It is a single-page app built using modern React features (e.g. hooks) in a node.js environment. It provides users the ability to log in, create and mofify posts, follow other users, search for posts by key words, view their own and others' profiles, and enter the site chat room. The site is reasonably, though I wouldn't say perfectly, responsive.

The above were developed in the course of following various tutorials on Udemy, where since the goal was to teach the specific technologies, much of the html and css was already provided, so I am responsible for only parts of their visual layouts.
